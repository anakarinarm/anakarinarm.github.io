2.4 En resumen¶

  • Importa bibliotecas usando import nombre-de-la-biblioteca.
  • Usa la biblioteca numpy para trabajar con arreglos en Python.
  • La expresin arreglo.shape te da las dimensiones del arreglo.
  • Usa arreglo[x, y] para seleccionar un elemento de un arreglo 2D.
  • Los ndices de los arreglos empiezan en 0, no en 1.
  • Usa la notacin min:max para especificar una "rebanada" que incluya desde el ndice min hasta max-1.
  • Usa # algo de texto para agregar comentarios en tus programas.
  • Usa numpy.mean(arreglo), numpy.max(arreglo) y numpy.min(arreglo) para calcular estadstica simple.
  • Usa numpy.mean(arreglo, axis=0) o numpy.mean(arreglo, axis=1) para calcular promedios sobre un eje especfico.

3.1 Grupos de gráficas¶

Podemos agrupar grficas similares en una sola figura usando subgrficas. El script o progama siguiente usa varios comandos que no hemos visto. La funcin plt.figure()crea el espacio para una figura vaca en donde pondremos nuestras grficas. El parmetro figsizele dice a Python de qu tamao ser la figura. Cada subgrfica se colocar en la figura usando el mtodo add_subplot. add_subplot toma 3 parmetros. El primero denota cuntas filas de grficas habr en total, el segundo cuntas columnas y el ltimo denota que a subgrfica hace referencia nuestra variable (derecha a izquierda y de arriba hacia abajo). Cada subgrfica se guarda en una variable distinta (axes1, axe2s, axes3). Una vez que se crea una subgrfica, se puede agregar ttulo y numbres a los ejes usando set_xlabel(),set_ylabel(), etc.

In [46]:
# Este es un script completo - una unidad que importa las bibliotecas,
# lee los datos del archivo txt, calcula estadstica y 
# grafica los resultados

import numpy
import matplotlib.pyplot

data = numpy.loadtxt(fname='precip_mensual_estacion01.csv', delimiter=',')

fig = matplotlib.pyplot.figure(figsize=(10.0, 3.0))

axes1 = fig.add_subplot(1, 3, 1)
axes2 = fig.add_subplot(1, 3, 2)
axes3 = fig.add_subplot(1, 3, 3)

axes1.set_ylabel('promedio')
axes1.plot(numpy.mean(data, axis=0))

axes2.set_ylabel('max')
axes2.plot(numpy.max(data, axis=0))

axes3.set_ylabel('min')
axes3.plot(numpy.min(data, axis=0))

fig.tight_layout()

matplotlib.pyplot.savefig('precipitacion.png')
matplotlib.pyplot.show()

fig.tight_layout() le da espacio suficiente a las grficas dentro de la figura para que no queden aplastadas. La llamada a savefig guarda la figura como un archivo grfico que se puede usar en otros documentos, por ejemplo para ponerlo en sus reportes ;-)

El formato del archivo lo determina matplotlib automticamente a partir de la extensin en el nombre que le dimos al archivo. En el programa anterior es PNG porque usamos el nombre 'precipitacin.png', pero se puede usar PDF, SVG, JPG, EPS, etc.
