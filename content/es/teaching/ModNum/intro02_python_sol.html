Verdadero y Falso¶

Las palabras True (verdadero) y False (falso) son especiales en Python. Se llaman booleanos y representan la "verdad" en los valores. Una expresin como 1<0 regresa el valor False, mientras que -1<0regresa el valor True.

Empecemos definiendo una funcin fahr_a_celsius que convierte temperaturas en escala Fahrenheit a Celsius:

In [41]:
def fahr_a_celsius(temp):
    return((temp - 32) * (5/9))

La definicin comienza con def seguido del nombre que queremos darle a la funcin (fahr_a_celsius) y entre parntesis una lista de nombres de parmetros (temp). El cuerpo de la funcin - los comandos que se ejecutan cuando corre la funcin - deben estar indentados. El cuerpo concluye con return seguido del valor que regresar la funcin.

Cuando llamamos a la funcin, los valores que le pasamos de le asignan a las variables que se usarn dentro de la funcin. Dentro de la funcin, usamos la expresin return regresar el resultado a quien lo pidi.

Corramos nuestra funcin:

In [42]:
fahr_a_celsius(32)
Out[42]:
0.0

Este comando llama a la funcin, usando "32" como input y regresa el valor de la funcin.

Llamar a nuestra funcin es lo mismo que llamar a cualquier funcin que hemos usado antes:

8.1 Composición de funciones¶

Ahora escibamos una funcin que transforme de Celsius a Kelvin:

In [44]:
def celsius_a_kelvin(temp_c):
    return temp_c + 273.15

print('El punto de fusin del agua en Kelvin:', celsius_a_kelvin(0.))

Qu hay de convertir de Fahrenheit a Kelvin? Podramos escribir otra funcin pero no es necesario. Podemos componer dos funciones (como en su clase de matemticas) que ya creamos:

In [45]:
def fahr_a_kelvin(temp_f):
    temp_c = fahr_a_celsius(temp_f)
    temp_k = celsius_a_kelvin(temp_c)
    return temp_k

8.3 En resumen¶

  • Define un funcin usando def nombre_de_funcin(parmetros)
  • El cuerpo de la funcin debe estar indentado.
  • Llama a una funcin usando nombre_de_funcin(valor)
  • Las variables definidas dentro de un a funcin solo pueden ser vistas y usadas dentro de esa funcin.
  • Si una variable no est definida dentro de la funcin, Python buscar la definicin dentro del cdigo que se encuentra antes de la llamada a la funcin.
  • Usa help(algo) para ver la ayuda de algo.
  • Agrega texto para documentar tu funcin usando ''' texto '''.
  • Especifica valores por default para los parmetros usando parmetro=valor default en la lista de parmetros dentro de la definicin de la funcin.
  • Los parmetros se pueden pasar a una funcin basados en el nombre, la posicin u omitindolos, en cuyo caso se usa el valor default.

Ambos graficos tienen similitudes y diferencias. Pcolormesh genera una funcin que asigna un color, dentro de un mapa de colores, al rango de datos dentro de los cuales se encuentran los valores del arreglo. Dependiendo del valor del elemento se le asigna un color. En cambio, contourf genera contornos, como curvas de nivel, para los valores del arreglo. T puedes elegir el nmero de contornos a graficar y, para ambas funcones, el mapa de color que quieras utilizar.
